dropShortIntervals

PURPOSE

Drops the intervals in the set that are shorter than a given threshold

SYNOPSIS

function iso = dropShortIntervals(is, thr, varargin)

DESCRIPTION 

  Drops the intervals in the set that are shorter than a given threshold
  
      USAGE:
      iso = dropShortIntervals(is, thr, OptionName, OptionValue) 
      
      INPUTS:
      is  - an intervalSet
      thr - a duration threshold; all intervals shorter than thr will be
            dropped
      
      OUTPUTS:
      iso - the resulting intervalSet

CROSS-REFERENCE INFORMATION

  • Start Starting times of for each interval in the set
  • intervalSet Constructor for intervalSet objects

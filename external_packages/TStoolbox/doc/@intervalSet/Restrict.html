PURPOSE

Returns subsets of an intervalSet to which timestamps belong to

SYNOPSIS

function [S,ix] = Start(O,tsa)

DESCRIPTION 

  Returns subsets of an intervalSet to which timestamps belong to
  
      USAGE:
      S = Restrict(O, TimeUnits)
      
      INPUTS: 
      O   - an intervalSet object

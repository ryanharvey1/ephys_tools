mergeCloseIntervals

PURPOSE

Merge together intervals that are closer than a given threshold

SYNOPSIS

function iso = mergeCloseIntervals(is, thr, varargin)

DESCRIPTION 

  Merge together intervals that are closer than a given threshold
  
      USAGE:
      iso = mergeCloseIntervals(is, thr, OptionName, OptionValue) 
      
      INPUTS:
      is  - an intervalSet
      thr - a distance threshold; if two consecutive intervals are closer than
            thr they will be merged 
      
      OUTPUTS:
      iso - the resulting intervalSet

CROSS-REFERENCE INFORMATION

This function calls:
  • dropShortIntervals Drops the intervals in the set that are shorter than a given threshold
  • length Returns the length of each interval in the set
  • timeSpan Returns the minimum continuous interval that contains the intervalSet

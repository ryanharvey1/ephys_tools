Range

PURPOSE

Returns TSD timestamps

SYNOPSIS

function R = Range(tsa, tsflag)

DESCRIPTION 

 Returns TSD timestamps
 
  USAGE:
  R = Range(tsa)
  R = Range(tsa, tsflag)
  
  OPTIONS:
  tsflag - if 'ts' returns time in timestamps (default),
          if 'sec' returns time in sec
          if 'ms' returns time in ms

instantRate

PURPOSE

Computes instantaneous rate as the inverse of inter-event interval

SYNOPSIS

function ir = instantRate(tsa, TimeUnits)

DESCRIPTION 

  Computes instantaneous rate as the inverse of inter-event interval
      
      USAGE:

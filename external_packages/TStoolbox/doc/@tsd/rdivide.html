rdivide

PURPOSE

Overload of the ./ operator for two tsd objects

SYNOPSIS

function s = rdivide(a,b)

DESCRIPTION 

  Overload of the ./ operator for two tsd objects
      
      USAGE:
      tso = rdivide(tsa,tsb) 
      
      the ./ operaetor is defined to return a tsd with same timestamps and
      data that are the ratio of the data. Returns error if timestamps of the
      two object don't coincide

CROSS-REFERENCE INFORMATION

This function calls:
  • compatible_ts Tells whether two tsd's have the same timestamps, within relative tolerance
  • tsd Constructor of TSD object

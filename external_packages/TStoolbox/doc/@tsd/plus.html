plus

PURPOSE

Overload of the + operator

SYNOPSIS

function s = plus(a,b)

DESCRIPTION 

  Overload of the + operator 
  
      USAGE:
      s = plus(a,b)
      
      the + operaetor is defined to return a tsd with same timestamps and
      data that are the sum of the data. Returns error if timestamps of the
      two object don't coincide. One of the two operands may be a scalar

CROSS-REFERENCE INFORMATION

This function calls:
  • compatible_ts Tells whether two tsd's have the same timestamps, within relative tolerance
  • size Return the size of the data component (overoad of the size function)
  • tsd Constructor of TSD object

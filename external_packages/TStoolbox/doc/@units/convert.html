convert

PURPOSE

c = convert(A, B) returns multiplicative factor to convert unit A into unit B

SYNOPSIS

function c = convert(A, B)

DESCRIPTION 

 c = convert(A, B) returns multiplicative factor to convert unit A into unit  B

 the conversion is such that if a is data expressed in unit A
 b = c * a is the same data expressed in unit B
